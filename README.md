# AaTempSpoof 功能介绍

AaTempSpoof 是一个面向 Android Root 设备的温度伪装、去温控、充电优化和性能辅助模块。模块主要用于降低系统读取到的温度数据、解除部分温控限制，并通过 WebUI 提供可视化调节。

## 主要功能

- 温度伪装：支持 CPU、GPU、DDR/内存、电池、机身外壳和其他温度节点伪装。
- 动态温度伪装：可设置固定温度，也可设置温度区间，让伪装温度在区间内动态变化。
- 电池温度伪装：支持单独伪装电池温度，降低系统、应用或充电策略读取到的电池温度。
- 充电过程温度伪装：支持设置充电开始温度、中间温度、截止温度和伪装时长，让充电时温度按设定曲线变化。
- 电池循环伪装：支持伪装电池循环次数。
- 电量显示伪装：支持伪装当前电量显示。
- 温度墙处理：支持提高或修改部分热区温度墙，GPU/DDR 温度墙可通过开关控制。
- 全机型去温控：通过热控配置覆盖、thermal governor 处理和 cooling device 状态清理，减少系统温控限频影响。
- 充电限制解除：处理慢充、夜间充电、灭屏降速、智能充电、充电保护、热控限流等限制。
- 快充协议辅助：支持 VOOC、SVOOC、UFCS、PD/PPS 等充电场景下的状态识别与辅助处理。
- 充电插拔功能：支持模拟插拔、协议判断、定时触发和充电状态伪装。
- 欧加 horae 控制：支持充电时禁用 horae，也支持通过白名单控制模块总开关。
- 前台应用白名单：可根据当前前台应用自动开启或关闭模块总开关。
- 触控采样率调节：支持通过 `touchHidlTest` 调整部分机型的触控采样率。
- ColorOS 墓碑相关处理：支持通过开关管理部分 ColorOS/OPlus 墓碑策略。
- HW 叠加层控制：支持一键禁用或恢复 HW 叠加层。
- 通知功能：支持模块启动通知、通知测试和自定义图片通知。
- WebUI 控制台：支持查看运行状态、实时温度、电池信息、配置开关、日志和附加功能。
- 极简模式：刷入即用，无需 WebUI 配置，自动启用基础温度伪装和去温控。
- 高级模式：可通过 WebUI 自定义温度、电池、充电、插拔、白名单、通知和其他扩展功能。
- 快捷开关 APK：可选安装控制中心快捷开关，用于快速控制模块总开关。
- 守护进程保活：开机后自动启动主进程和辅助进程，并通过 watchdog 检测和重启异常进程。
- 卸载恢复：卸载时会停止相关进程，清理挂载和运行文件，并尽量恢复热控、充电、电池相关状态。

## WebUI 可调项目

- 模块总开关。
- CPU/GPU/内存/电池/其他温度伪装开关。
- 各类温度伪装数值。
- 动态温度区间。
- 电池循环次数。
- 当前电量显示。
- 电池充电温度曲线。
- 温度墙开关。
- 充电插拔模式。
- 插拔触发电量和间隔。
- horae 开关与白名单。
- 前台应用白名单。
- 触控采样率。
- 通知开关。
- 旁路、兼容模式、开发者模式等测试功能。

## 适用场景

- 想降低系统或应用读取到的温度显示。
- 想减少游戏、充电或高负载场景下的温控限制。
- 想改善灭屏充电、夜间充电、慢充、温控限流等问题。
- 想通过 WebUI 精细控制温度伪装和充电相关行为。
- 想用快捷开关快速开启或关闭模块。

## 注意事项

- 需要 Android Root 环境。
- 更适合 arm64、内核 5.10 及以上设备。
- 不同机型节点不同，部分功能可能只在特定品牌或系统上生效。
- 不建议和其他温度伪装、去温控、充电加速类模块同时使用。
